#  Build Data Infra

## Descripción del Problema

El día a día de los sistemas actuales es crear logs de las actividades de los usuarios mientras hacen uso de los productos, para entender el comportamiento de los usuarios y potenciar los sistemas de recomendación.


## Preguntas

Construye una infraestructura de data que sporte las siguientes necesidades

1. **Task:** Logs de event data de una aplicación mobile y los servicios web.
2. **Task:** Persite la data en un storage offline.
3. **Task:** Cataloga la data y presentala como una tabla estructurada de tal manera que pueda ser analizada en SQL.
4. **Task:** Construye la infraestrutura para transformar/analizar y presentar la data en un dashboard BI asi como generacion de reportes.
5. **Task:** Provee buena escalabilidad, fiabilidad(reliability) y efficiendia además como usabilidad en los sistemas.
